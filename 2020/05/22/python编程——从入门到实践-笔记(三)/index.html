将函数储存在模块中

所有的import语句都应放在文件开头，注释除外

模块是扩展名为.py的文件

  • 导入整个模块：

    （函数在A文件中）在主文件中：

    使用： A.函数名(参数。。。)

  • 导入特定的函数

    from 文件名 import 函数1，函数2，函数3

  • 使用：函数（参数列表） *

  • 使用as给模块指定别名

    import 文件A as A
    使代码更简洁

  • 导入模块中的所有函数

    from 文件A import*
    使用：直接使用函数，无需点号
    不建议使用，防止模块函数名与自己编写的函数名重名：重名覆盖函数
