1.2 相关的汇编指令

  • 在32位操作系统中，出，入栈时bsp每次改变4字节
  • 某些指令会自动地操作堆栈:call,ret

    call本质上是：push+jmp；ret本质上是：pop+jmp

  • sub,add也可用于操作栈

    sub:一次性分配栈空间；add：用于分配函数局部变量空间

  • 异或XOR：XOR eax，eax；代替 mov eax 0；速度更快 ，占用字节更少。
  • lea常用于赋值操作:lea edi [ebp-0cch]代替不支持的mov edi [ebp-0cch]
