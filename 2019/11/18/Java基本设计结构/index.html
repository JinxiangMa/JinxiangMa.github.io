简单程序

Java的全部内容放置在类中

类名的命名格式：驼峰

源文件的文件名与公共类必须相同

  • 在类的源文件必须包含main方法
  • println在控制台输出字符串，并终止该行，每次调用在新的一行输出（print不换行）

注释

/* 开始。。。。结束/——自动生成文档

数据类型

  • 整型：
    • int（4字节）
    • short（2字节）
    • long（8字节）：加后缀L
    • /1byte（1字节）
  • 整型的范围与具体的机器或操作系统无关*——移植性

    注意：Java中没有无符号整型的概念

  • double（后缀D）的数值精度是float（后缀f）的两倍

    无后缀的浮点数默认为double类型

  • 表示溢出和出错的三个浮点数值：
    • 正无穷大：Double.POSITIVE_INFINITY
    • 负无穷大：Double.ENGATIVE_INFINITY
    • NAN（不是一个数字）:Double.NAN

char类型

表示：’A’
范围：\u000~\Ufff(转义序列\u)
Unicode转移序列会在解析代码之前得到处理

  • 当心注释中的\u:\u00A0为换行符——将注释变为代码
  • 注释中的\users会被识别为语法错误：不能转义

变量

  • 变量名是字母开头，包含数字字母的序列
  • 不要在自己的代码中使用“$”字符，它只用在Java编译器和其他工具生成的名字中*
