实例二：用IDC分析加密算法–encrypted.exe程序

注意：对有疑问的地方IDA会用红色显示。
SMC技术：可执行文件中保存着加密数据，只有在执行时才会在某处用还原你代码解密。

//encrypted.idc
//(c)  www.PEDIY.com 2000-2008

#include <idc.idc>
static decrypt(from, size, key ) {     //from加密起始地址 
   auto i, x; 
   for ( i=0; i < size; i=i+1 ) { 
      x = Byte(from);      //取得解密数据
      x = (x^key);            //异或解密
      PatchByte(from,x); //将数据还原
      from = from + 1;
   } 

}
