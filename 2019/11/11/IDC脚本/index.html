IDC脚本

2019年11月11日 536 字 大概 2 分钟
类C的嵌入式语言，在特殊情况的控制下，自动化完成IDA重复工作。

实例一：查看输入函数

原理：通常，某些编译器给输入表所在区段生成的默认名字为.idata，仅为实验考虑。实际情况要考虑PE文件的结构特征。

参考书籍：《IDA Pro权威指南》

实例二：用IDC分析加密算法–encrypted.exe程序

注意：对有疑问的地方IDA会用红色显示。
SMC技术：可执行文件中保存着加密数据，只有在执行时才会在某处用还原你代码解密。

//encrypted.idc
//(c)  www.PEDIY.com 2000-2008

#include <idc.idc>
static decrypt(from, size, key ) {     //from加密起始地址 
   auto i, x; 
   for ( i=0; i < size; i=i+1 ) { 
      x = Byte(from);      //取得解密数据
      x = (x^key);            //异或解密
      PatchByte(from,x); //将数据还原
      from = from + 1;
   } 

}
